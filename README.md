# Wedding Invitation Website using AI 🌸

A premium, interactive, and beautifully animated wedding invitation website built with Next.js, TailwindCSS v4, and Framer Motion. 

This project was built to celebrate the wedding of **Jahnavi & Penchala Swamy**.

## 🌟 Features

- **🌸 Interactive welcome screen**: Engaging welcome interface to build excitement.
- **🎵 Atmospheric music player**: Background music player loaded with custom wedding waltz.
- **✨ Particle animations**: Floating cherry blossom petals floating gracefully across the screen using framer-motion.
- **📅 Detailed Event Schedule**: Multiple events with timings, locations, and beautiful typography.
- **📍 Interactive Venue map**: Structured venue details with navigation links.
- **👨‍👩‍👧 Family Section**: Showcases family members and their blessings.
- **📞 Contact & RSVP**: Quick contact links for coordinators and RSVP details.
- **📱 Fully Responsive Design**: Seamless premium experience on mobile, tablet, and desktop screens.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: TailwindCSS v4
- **Animations**: Framer Motion
- **Language**: TypeScript

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```
